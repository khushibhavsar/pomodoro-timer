*What is the Pomodoro Timer?* The Pomodoro Timer is a productivity app that allows a user to set a time to allot for their studies or tedious tasks. 

*Why was this project made?* The pomodoro timer was made in mind for individuals who are unable to focus and need an external reminder. The app incorporates multiple features that help the process and make it enjoyable for the user. 

*How was this project made?* This project was made using HTML, CSS, and Javascript. I used a variety of sources for instance; an article from geeksforgeeks.org that helped me with the javascript portion of the app. I also watched online videos that guided me specific features in the app, for instance; how to add music, how to have a day/night mode, and how to add a progress ring. 

*What you stuggled with + what you learned.* This project taught me how to implement timers using setInterval and clearInterval, which helped me understand how to manage time-based logic and avoid issues like multiple intervals running at once. I also learned how to think in terms of state. For example, keeping track of whether the timer is paused, whether I’m in study or break mode, and how those states affect what the app should do next. That part really helped me understand how real applications manage logic, not just single functions. It was also my first time working with browser APIs such as localStorage to add dark mode settings and the Notification API to display alerts.
