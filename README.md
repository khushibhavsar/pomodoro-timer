__*What is the Pomodoro Timer?*__ The Pomodoro Timer is a productivity app that allows a user to set a time to allot for their studies or tedious tasks. 

__*Why was this project made?*__ The pomodoro timer was made in mind for individuals who are unable to focus and need an external reminder. The app incorporates multiple features that help the process and make it enjoyable for the user. 

__*How was this project made?*__ This project was made using HTML, CSS, and Javascript. I used a variety of sources for instance; an article from geeksforgeeks.org that helped me with the javascript portion of the app. I also watched online videos that guided me specific features in the app, for instance; how to add music, how to have a day/night mode, and how to add a progress ring. 

__*What you stuggled with + what you learned.*__ A common mistake I made in this project was making sure I was adding the correct name of the function as any minor change was enough to make the code to stop working. There was also the feature of adding a progress bar that was something new I learned as it was actively monitoring the time and changing likewise.
